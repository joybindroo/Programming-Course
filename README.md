# A Comprehensive Computer Science & Programming Course: From Novice to Expert

> **Live site:** https://joybindroo.github.io/Programming-Course/

## Modules Overview

### Module 1: Foundations of Computer Science
- **Understanding Computers** – How computers work, binary number system
- **Introduction to Algorithms** – Definitions, problem‑solving techniques, flowcharts & pseudocode
- **Data Structures** – Arrays, lists, stacks, queues and their relationships

### Module 2: Programming Fundamentals
- **Variables & Data Types** – Primitive (int, float, char, bool) and complex (string, array)
- **Control Flow** – Conditionals (if‑else, switch) and loops (for, while, do‑while)
- **Functions** – Definition, parameters, return values
- **Input & Output** – Reading user input and printing output

### Module 3: Object‑Oriented Programming (OOP)
- **Core Concepts** – Classes, objects, encapsulation, inheritance, polymorphism
- **Designing OOP Systems** – Modeling problems, identifying objects, class diagrams

### Module 4: Data Structures and Algorithms
- **Arrays & Strings** – Searching, sorting, manipulation
- **Linked Lists** – Singly & doubly linked lists, insertion/deletion/traversal
- **Stacks & Queues** – Operations and applications
- **Trees & Graphs** – Traversal algorithms (BFS, DFS), binary trees, graphs
- **Recursion** – Techniques and classic examples (factorial, Fibonacci, Tower of Hanoi)

### Module 5: Advanced Topics
- **File I/O & Data Pipelines** – Streaming text/binary, schema validation, atomic writes
- **Defensive Exception Handling** – Custom hierarchies, retries, structured logging
- **Concurrency & Parallelism** – Threads, processes, async IO, coordination primitives
- **Design Patterns** – Creational, structural, behavioral patterns with trade‑offs
- **Web Development Deep Dive** – REST services, secure APIs, responsive dashboards, CI observability
- **Mobile App Development** – Native vs cross‑platform, offline sync, telemetry
- **Game Development Foundations** – Game loops, ECS/state machines, physics, asset pipelines
- **Machine Learning & AI** – Data prep, model training/evaluation, deployment, drift monitoring
- **Systems Integration Capstone** – Combine web, mobile, ML, realtime into a documented end‑to‑end project

### Module 6: Software Delivery & DevOps
- **Release Engineering & CI/CD** – Branching strategies, quality gates, artifact versioning
- **Infrastructure as Code** – Terraform/Pulumi patterns, secrets hygiene, environment parity
- **Containers & Orchestration** – Docker best practices, Kubernetes/serverless workflows
- **Observability & Incident Response** – Metrics, logs, traces, alerting, on‑call playbooks
- **Security & Compliance** – Shift‑left scans, policy‑as‑code, audit readiness
- **Resilience & Progressive Delivery** – Blue/green, canary, chaos engineering, feature flags
- **Delivery Capstone** – Ship a production‑ready version of the Module 5 project with CI/CD, IaC, telemetry, and rollback plans

### Module 7: Docker Mastery
- **Installation & Verification** – `docker version`, `docker info`, registry auth, health checks
- **Context Management** – Local/SSH/cloud contexts for remote control
- **Image & Registry Operations** – Build, pull/push, tag, save/load, SBOM‑friendly pipelines
- **Container Lifecycle & Observability** – Run, ps, stop/start, logs, stats, prune
- **Storage & Networking** – Networks, volumes, custom bridges
- **Compose & Multi‑Service Apps** – `docker compose` up/down, scaling, rebuilding
- **Security & Distribution** – `docker scan`, least‑privilege flags, artifact shipping
- **Command Checklist** – 30+ essential Docker CLI commands from installation to production

### Module 8: Database Programming & Tooling
- **Multi‑Engine Setup** – PostgreSQL, MySQL, SQLite via packages or Docker Compose
- **Schema & Migration Discipline** – Normalization, constraints, versioned migrations
- **PostgreSQL Focus** – `psql` workflow, `EXPLAIN ANALYZE`, indexes, `psycopg` examples
- **MySQL Focus** – `mysql` CLI, transaction tuning, `mysql‑connector‑python`
- **SQLite Focus** – Shell commands, WAL mode, built‑in Python driver
- **DBeaver Mastery** – Visual schema browsing, ER diagrams, secure connections
- **Python Data Layers** – Parameterized queries, connection pools, async drivers, SQLAlchemy
- **Capstone** – Multi‑database CRM with shared models, migrations, and DBeaver assets

### Module 9: Distributed Systems
- **Fundamentals** – Scalability, consistency, fault tolerance
- **Key Technologies** – Consensus algorithms, message queues, microservice patterns
- **Practical Labs** – Replicated key‑value store, Docker Compose deployment
- **Capstone Project** – Resilient multi‑node application with health checks and auto‑recovery

### Module 10: AI Application Development
- **Overview** – Build intelligent apps with Python using LLM APIs (OpenAI, Gemini, OpenRouter, Ollama)
- **Key Topics** – API authentication, prompt engineering, streaming responses, function calling, retrieval‑augmented generation, multi‑model routing
- **Integration** – FastAPI/Flask, Docker containerization, local Ollama deployment
- **Hands‑On Labs** – Chat bot, Gemini switch, OpenRouter comparison, Ollama service

### Module 11: Fine‑Tuning Large Language Models with Unsloth
- **Overview** – Adapt pre‑trained LLMs to specific domains or styles using LoRA adapters and fast training loops
- **Key Topics** – When to fine‑tune, why Unsloth (speed, low VRAM, simple API), dataset preparation, training, deployment
- **Hands‑On Lab & Checklist** – End‑to‑end fine‑tuning workflow

---

By following this comprehensive course you will develop a solid foundation in computer science and programming, enabling you to tackle complex problems and build innovative solutions.
